# 设置篇

- ## 更改IntelliJ IDEA生成的注释的user的默认值

1、file -->  settings --> File and Code Templates
2、中间的，选择 includes ，选中File Header。
3、右侧会出现 Created by ${USER} on ${DATE}.  
4、把这个 ${USER} 直接替换掉 Created by myname on ${DATE}.
